## What is Machine Learning?

Machine Learning (ML) is a branch of artificial intelligence that enables computers to learn from examples rather than being explicitly programmed. Think of it like teaching a child:

- **Traditional Programming**:

  ```
  IF (sees_round_orange_object) AND (has_bumpy_texture) THEN "it's an orange"
  ```

- **Machine Learning**:
  Show many examples of oranges and non-oranges, and let the computer learn the patterns

## How Computers "See" Images

When computers process images, they don't "see" like we do. Instead, they work with a grid of numbers representing pixel values. Each pixel contains information about color intensity, typically in RGB (Red, Green, Blue) format. For example, a bright red pixel might be represented as [255, 0, 0], while a dark blue would be [0, 0, 100]. Machine learning models analyze these numerical patterns to identify objects and features within images.

1. **Pixels**

   - Images are grids of numbers
   - Each pixel has color values (RGB)
   - Example:
     ```
     Red pixel: [255, 0, 0]
     Green pixel: [0, 255, 0]
     Blue pixel: [0, 0, 255]
     ```

2. **Features**

Features are the building blocks that machine learning models use to understand images. At the most basic level, they start with patterns of pixels - recurring arrangements of color values that might indicate something meaningful. The model then identifies key visual elements like edges where colors change sharply, corners where edges meet, and textures that show repeating patterns.

It also analyzes colors and shapes, looking for distinctive hues and geometric forms that could help identify objects. These simpler features are then combined into more complex patterns - for example, several edges might form the outline of an object, while combinations of colors and textures could indicate specific materials or surfaces.

### From Pixels to Predictions

1. Image input ‚Üí Grid of numbers
2. Find important features
3. Learn which features match which classes
4. Make predictions based on features
