# Getting Started with Machine Learning

Before we dive into the exciting world of machine learning, let's ensure you have everything needed for a smooth workshop experience.

## Prerequisites

### Hardware Requirements

- A computer (laptop or desktop)
- Working webcam (built-in or external)
- Stable internet connection
- Modern web browser (Chrome recommended)

### Software Setup

1. **Browser Check**

   - Open [Chrome](https://www.google.com/chrome/) or any other modern web browser
   - Ensure your browser is up to date
   - Allow camera permissions when prompted

2. **Teachable Machine Access**
   - Visit [Teachable Machine](https://teachablemachine.withgoogle.com)
   - No account required
   - No installation needed

## Initial Setup

### 1. Creating Your First Project

1. Navigate to Teachable Machine:

   ```
   https://teachablemachine.withgoogle.com
   ```

2. Click the "Get Started" button

![Image](/screenshots/step-1.png)

3. Select "Image Project" as the project type

![Image](/screenshots/step-2.png)

4. Click "Standard image model"

![Image](/screenshots/step-3.png)

5. This creates a new blank project

![Image](/screenshots/step-4.png)

### 2. Understanding the Interface

The Teachable Machine interface has several key areas:

1. **Class Panel** (Left)

   - Where you'll create and manage classes
   - Each class represents a category to recognize

2. **Training Panel** (Right)
   - Contains the "Train Model" button
   - Shows training progress and stats

## Testing Your Setup

Before proceeding, let's do a quick test:

1. **Camera Test**

   - Click the "Webcam" button under Class 1
   - You should see your camera feed
   - If not, check browser permissions

![Image](/screenshots/step-5.png)

2. **Interface Test**
   - Try renaming Class 1 to "Test"
   - Capture a few sample images (when you press hold to record, it will burst capture images from your web cam)
